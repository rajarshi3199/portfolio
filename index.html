IPL-win-probability-predictor

Web App

I've developed a machine learning model to predict the winning team in Indian Premier League (IPL) cricket matches. The model utilizes a combination of categorical and numerical features, including the batting team, bowling team, and city where the match is played. To handle the categorical data, I've implemented a preprocessing pipeline using scikit-learn's ColumnTransformer and OneHotEncoder. This transforms the categorical variables into a format suitable for machine learning algorithms. The core of the model is a Logistic Regression classifier, chosen for its interpretability and efficiency in binary classification tasks. I've integrated this into a scikit-learn Pipeline, which streamlines the preprocessing and model training steps. This approach allows the model to learn patterns from historical IPL data and make predictions on future matches, potentially offering valuable insights for cricket enthusiasts and analysts.
